# PomodoroApp

Una aplicación web de Pomodoro diseñada para ingenieras, con un estilo femenino, motivacional y profesional. Permite gestionar sesiones de estudio o trabajo usando la técnica Pomodoro, visualizar el progreso en un calendario y mantener un historial de sesiones.

## Características

- Temporizador Pomodoro ajustable
- Mensajes motivacionales personalizados
- Calendario visual con días marcados si estudiaste
- Historial de sesiones recientes
- Interfaz moderna, intuitiva y atractiva
- Accesible desde PC, tablet o móvil en la red local

## Instalación y uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/alejandrorodm/PomodoroApp.git
   cd PomodoroApp
   ```
2. Instala las dependencias:
   ```bash
   pip install flask
   ```
3. Ejecuta la aplicación:
   ```bash
   python app.py
   ```
4. Accede desde tu navegador a:
   - `http://localhost:9001` (en el mismo equipo)
   - `http://<IP_LOCAL>:9001` (desde otros dispositivos en la red)

## Personalización
- Puedes cambiar el puerto editando la línea correspondiente en `app.py`.
- El diseño y los mensajes pueden adaptarse fácilmente en `templates/index.html`.
